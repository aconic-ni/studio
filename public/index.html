Bienvenido a CustomsEX-p Assistant

Esta es una página HTML estática de marcador de posición.

La aplicación Next.js se inicia normalmente usando el comando npm run dev en su terminal y es accesible en el puerto configurado (por ejemplo, http://localhost:9002).

Este archivo index.html se sirve desde el directorio public y puede ser accedido en /index.html cuando el servidor de desarrollo está en ejecución.
