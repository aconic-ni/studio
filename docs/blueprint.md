# **App Name**: CustomsEX-p Assistant

## Core Features:

- Product Entry Form: Streamlined data entry for customs declaration items, including item number, description, brand, quantity, and condition.
- Real-time Validation: Real-time validation of data inputs to ensure accuracy and completeness of customs declaration information.
- HS Code Suggestion: AI-powered tool for suggesting harmonized system (HS) codes based on product descriptions to ensure compliance with customs regulations.
- Report Generation: Generates and downloads reports of entered data for submission to customs authorities, with options for TXT and Excel formats.

## Style Guidelines:

- Primary color: Slate blue (#7395AE) evoking reliability.
- Background color: Light gray (#D4DADF), creating a calm and neutral backdrop.
- Accent color: Soft, muted blue (#96B3C2), emphasizing interactivity and highlighting key elements.
- Clean, sans-serif font for optimal readability and a modern feel.
- Simple, clear icons that are easily recognizable, aiding navigation.
- Clean and efficient layout that minimizes clutter.
- Subtle transitions and loading indicators that enhance user experience without being distracting.
