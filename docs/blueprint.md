# **App Name**: Customs Ex-p

## Core Features:

- Initial Exam Form: Enable users to input general info to identify the exam.
- Products table: A product list showing added products as rows in a table.
- Add product form: Ability to add a product by specifying its features using a form.
- Preview data modal: A preview modal to show all the information before submitting.
- Report Generation: Ability to generate and download reports in TXT or Excel formats.

## Style Guidelines:

- Accent color: #63B3ED (sky blue) to complement the existing color scheme.
- Use the default system fonts for a clean and universal look.
- Use simple, outlined icons from a consistent set (e.g., Heroicons) for UI elements.
- Maintain a consistent grid-based layout for all sections.
- Subtle fade-in and slide-in animations for modals and transitions.
